# Outline by Heading and List

Here is an example for outlining via fusing heading and list.

- List item 1

  ## Heading 2

  A simple paragraph.

  > Blockquote
  >
  > - List item in blockquote
  >
  >   - Sub list item 1
  >   - Sub list item 2
  >

  - Sub list item 1
  - Sub list item 2
- ## List item 2

  1. Ordered list item 1

      ```go
      package main

      import "fmt"

      func main() {
      	fmt.Println("Hello, 世界")
      }
      ```
  2. Ordered list item 2
